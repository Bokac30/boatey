
Demo Setup Tetrapak

1. Download NodeJs and NPM:
	https://nodejs.org/dist/v10.9.0/node-v10.9.0-x64.msi
3. Extract the contents of the website (zip) in to a folder of your choice
4. Open the extracted folder and execute(double-click) the Start.bat file
5. If all steps are executed in order the website should open in a browser
	http://localhost:3000 or http://127.0.0.1:3000

Note: Use the Start.bat only onse when windows starts (or on restart)
The Start.bat file just starts a server to serve the website.
Then you can always open http://localhost:3000	to browse the website
